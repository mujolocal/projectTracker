###  Setup

```bash
python -m venv env
pip install -r requirements_file.txt

```

###  Run the Application

```bash
uvicorn main:app --reload
```

The application will start on `http://localhost:8000`

- **API Documentation**: `http://localhost:8000/docs`
- **Web Interface**: `http://localhost:8000`
